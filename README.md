# draw-a-shape
An Android game for testing how good your drawing skills are.

# About
Uses [draw-a-spape-algorithm](https://github.com/attrck/drawing-app-algorithm) via a web api. Tested with Samsung Galazy Note 10.1 2014 Edition.
